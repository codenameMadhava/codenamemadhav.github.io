What this project does

  • Runs a minimal Flask API service with a / route and /health endpoint
  • Packages the app into a Docker image (repeatable, portable runtime)
  • Deploys the image as a container using Docker Compose
  • Injects configuration using .env (no hardcoding config in code)
  • Exposes the service via host port mapping and verifies with curl
